# Jumia Pactical Task

## Installation:
- clone the repo `git clone https://github.com/AbdulRahmanAbouzaid/jumiaPactical.git`
- go to project folder `cd jumiaPactical`
- run `composer install`
- run `php -S localhost:8000` to serve the project.
